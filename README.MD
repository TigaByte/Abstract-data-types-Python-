## Phython Implementation of diffrent Abstract data types 

### Stack

**Functions**

| Function | Description |
|-----------|--------------|
| `stack.push(object)` | Pushes an object onto the stack. |
| `stack.pop()` | Removes the top object from the stack and returns its value. |
| `stack.peek()` | Returns the top value of the stack without removing it. |
| `stack.isEmpty()` | Returns `true` if the stack is empty; otherwise `false`. |
| `stack.size()` | Returns the number of elements in the stack. |
